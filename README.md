Contact Management Application

Project Summary
This application is a Java-based contact management system that allows users to add, update, delete, and display contacts.
This apllication uses SQLite database integration through JDBC to store and manage data.

Features
Add new contacts
Update existing contacts
Delete contacts
Display all contacts

Technologies Used
Java
SQLite
JDBC

How to Run
1. Open project in VS Code
2. Ensure SQLite JDBC is inlucded
3. Run Main.java
